Asyncio background tasks
========================

Asyncio background tasks in Python 3.7 and later.

Project homepage: https://github.com/eerimoq/asyncbg

Documentation: https://asyncbg.readthedocs.org/en/latest

Installation
============

.. code-block:: python

    pip install asyncbg
